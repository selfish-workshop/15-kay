# REPBOT (렙봇)

PT 없이 혼자 운동하는 사람을 위한 AI 운동 코칭 & 기록 서비스

## 프로젝트 정보
- **유형**: SaaS (웹앱/PWA)
- **타겟 사용자**: PT를 일부 경험했지만 이제 혼자 헬스장에서 운동하는 초심자~중급 사이 사용자
- **수익 모델**: 프리미엄 구독제
- **출시 목표**: 미정

## 핵심 기능
- 루틴 추천 + 개인 특수성 고려한 운동 가이드
- 루틴 수행 중 세트별/운동별 코멘트 작성
- 운동 전문가 데이터 기반 AI 피드백
- 코멘트 작성 시 과거 동일 운동 기록 + AI 피드백 실시간 표시 (실시간 교정 지원)
