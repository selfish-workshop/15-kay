# 기획 (Planning)

이 폴더는 "무엇을 만들 것인가"의 상세를 담습니다.

## 작성할 내용
- 기능 명세 (각 기능의 동작 정의)
- 사용자 시나리오 (Who, When, Where, Why, How)
- MVP 범위 (필수 vs 추후)
- 우선순위

## 서비스 흐름도

```
┌─────────────────────────────────────────────────────────────────┐
│                    REPBOT 서비스 흐름도                           │
└─────────────────────────────────────────────────────────────────┘

  ┌──────────────┐
  │  사용자 온보딩  │
  │  (신체정보,    │
  │   운동경험,    │
  │   목표 입력)   │
  └──────┬───────┘
         │
         ▼
  ┌──────────────┐
  │ ① 루틴 추천   │ ◄── AI 엔진: 개인 특수성 반영
  │  오늘의 운동   │     (부상이력, 체력수준, 목표)
  │  A. 벤치프레스  │
  │  B. 스쿼트     │
  │  C. 랫풀다운   │
  └──────┬───────┘
         │
         ▼
  ┌──────────────────────────────────────────────────────┐

  │  ┌─────────────────┐    ┌──────────────────────────┐ │

  │  │                  │    │  (코멘트 입력 시 자동 표시)  │ │
  │  │  벤치프레스 3세트   │    │                          │ │
  │  │  ────────────    │    │  ┌──────────────────────┐ │ │

  │  │  3세트: 65kg     │    │  │                      │ │ │
  │  │                  │    │  │ 🤖 AI 피드백:          │ │ │

  │  │  │ 오른쪽 어깨  │  │    │  │ 📅 1주 전 기록         │ │ │
  │  │  │ 약간 불편"   │  │    │  │ "55kg 5회, 무난"      │ │ │
  │  │  └────────────┘  │    │  │                      │ │ │
  │  │                  │    │  │ 🤖 AI 피드백:          │ │ │
  │  │                  │    │  │ "다음엔 60kg 도전"     │ │ │
  │  └─────────────────┘    │  └──────────────────────┘ │ │
  │                          └──────────────────────────┘ │
  └────────────────────────────┬─────────────────────────┘
                               │
                               ▼
                 ┌──────────────────────────┐
                 │ ③ AI 실시간 피드백          │
                 │                          │

                 │  반복되고 있습니다.          │
```
